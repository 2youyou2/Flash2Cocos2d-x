override public function playTo(_to:Object, _toFrames:uint, _listFrames:uint = 0, _loop:Boolean = false, _ease:int = 0):void

播放动画

Parameters

_to:Object — ArmatureAniData 在 ConnectionData 中的字符串索引
 
_toFrames:uint — 过渡到 _to 动画需要花费的帧
 
_listFrames:uint (default = 0) — _to 动画播放持续的帧
 
_loop:Boolean (default = false) — 是否循环
 
_ease:int (default = 0) — FrameNodeList 各个关键点的缓动方式，-1:SineOut，0:Line，1:SineIn，2:SineInOut
