_boneDisplayFactory:Function (default = null) — 骨骼显示对象的构建方法，如果不传递，在 stage 中将根据骨骼名，在骨架的显示对象中查询是否有符合这个名字的显示对象
 
_isRadian:Boolean (default = false) — 骨骼旋转角度是否采用弧度制，比如 Starling 使用的是弧度制
 
_useLocalXYZ:Boolean (default = false) — 是否使用骨骼现有的相对位置数据而不是 ConnectionData 中的
